Telegram Bot

Your best assistant!

All you need to do is create you own bot with help of the BotFather, install this application and restart odoo server with --load=telegram,web (check details on Documentation tab). Now you have your personal assistant:

There are three types of commands the bot can do:

  • Simple Request-Response commands
  • Notification commands
  • Periodic reports

Request-Response commands

Ask the bot what you need and he will reply immediately

Notification commands

The bot is ready to notify you about any events you don't want to miss.

Periodic reports

What about periodic reports, say, about new calls? The bot can send report on new created issues every hour.

Custom commands

This application provides base functionalities for telegram bot integration and few base commands. You can either create you own commands via Settings / Telegram menu or buy modules with prepared commands
